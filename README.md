# Sit With It

Landing page for the launch of Rose Attridge's book, **Sit With It**.

## Local preview

Open `index.html` directly in a browser, or run a tiny local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Kit hookup

The signup form in `index.html` is marked with `data-kit-placeholder`. Replace the `action="#"` form with the Kit embed or Kit form action when the mailing list is ready.

## Domain

When the domain is ready, add a `CNAME` file containing the production domain and configure the DNS provider for GitHub Pages.
